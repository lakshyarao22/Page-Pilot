# Privacy Policy for Page Pilot

Page Pilot is a Chrome new tab extension that provides bookmarks, todos, notes, widgets, wallpapers, weather, and productivity tools.

## Data Stored Locally

Page Pilot stores user preferences and widget data locally in the browser using localStorage. This may include bookmarks, todo items, quick notes, habit entries, countdowns, world clock settings, Pomodoro timer state, greeting name, widget layout preferences, wallpaper settings, and an optional Unsplash access key.

This data stays on the user’s device and is not transmitted to the developer.

## Location and Weather

If the user allows browser location access, Page Pilot uses the device’s approximate latitude and longitude to fetch current weather from Open-Meteo. This request is made directly from the user’s browser to Open-Meteo over HTTPS. Page Pilot does not store, sell, or share location data with the developer.

## Wallpapers

Page Pilot may load wallpaper images from Unsplash or Picsum Photos. If the user provides an Unsplash access key, it is stored locally in the browser and used only to request wallpapers from Unsplash.

## Notifications

Page Pilot uses browser notifications only to alert the user when a Pomodoro focus or break timer completes. Notification content is generated locally and is not sent to the developer.

## Data Sharing

Page Pilot does not sell user data, transfer user data to third parties for advertising, or use user data for tracking.

External services used by the extension:
- Open-Meteo: weather data
- Unsplash: optional wallpaper images
- Picsum Photos: fallback wallpaper images

## Security

All external requests made by Page Pilot use HTTPS.
